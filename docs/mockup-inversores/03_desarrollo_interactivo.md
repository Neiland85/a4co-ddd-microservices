## 03 Desarrollo Interactivo

- **Objetivo**: Construir el mockup navegable con tecnología web para demostrar funcionalidad y propuesta de valor.

### Actividades clave

- **Setup técnico**: Configurar repo demo (Next.js/React), linters y despliegue temporal (Vercel/Netlify).
- **Implementación de pantallas**: Traducir diseños a componentes reutilizables, asegurando responsividad.
- **Datos simulados**: Crear fixtures JSON y hooks que permitan alternar escenarios (éxito, crecimiento, métricas).
- **Microinteracciones**: Aplicar animaciones ligeras para reforzar fluidez y storytelling.
- **Copia y métricas**: Inyectar textos orientados a inversores y gráficos con cifras plausibles.
- **QA visual y funcional**: Revisar en navegadores clave y dispositivos para la demo.

### Entregables

- Repositorio del mockup con instrucciones básicas de ejecución.
- Build desplegada en entorno de pruebas.
- Checklist de QA completado.

### Riesgos y mitigación

- Desalineación diseño-dev → sesiones de handoff y uso de librerías de tokens compartidos.
- Performance deficiente → optimizar assets, lazy loading para dashboards.
- Demo inestable → scripts de seed y fallback offline para situaciones sin conexión.
