## Mockup Inversores

- **Objetivo**: Documentar el proceso de desarrollo de un mockup navegable para presentaciones con potenciales inversores.
- **Alcance**: Cobertura de fases de investigación, diseño, desarrollo, validación y entrega.
- **Estructura**: Cada archivo detalla actividades, entregables y responsables sugeridos por fase.

### Guía rápida

- `01_preparacion_estrategica.md`: Alineación inicial, insights clave y definición de alcance.
- `02_diseno_ux_ui.md`: Arquitectura de información, wireframes y diseño visual.
- `03_desarrollo_interactivo.md`: Implementación técnica del mockup y datos simulados.
- `04_validacion_y_demo.md`: Feedback interno, iteraciones y preparación de la demo.
- `05_documentacion_y_entrega.md`: Materiales finales y próximos pasos.

> Recomendación: complementar esta guía con el roadmap detallado de esfuerzos estimados para coordinar recursos y agenda.
